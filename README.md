# 3D Pyramid Landing Page
A dynamic 3D landing page using Three.js

## Front-end
* HTML, CSS (SASS), JavaScript. 
* Background animation: GSAP, Three.js. 
* Three.js: 
  - Scene, camera, renderer. 
  - Plane mesh, directional light. 
  - Modify with dat.gui. 
  - screen rotation: OrbitControl. 
  - Hover effect.
